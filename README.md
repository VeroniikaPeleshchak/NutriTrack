# NutriTrack

## Description
NutriTrack is an iOS app designed to help users track their daily nutrition, calories, and macronutrients (protein, fat, carbs) as well as other daily measurements such as weight, water intake, and activity. The app provides visual charts to monitor progress over time and simple recommendations to support healthy habits.

## Purpose
The main goal of NutriTrack is to make it easy and convenient for users to maintain a personal food diary, analyze their eating patterns, and track changes in their health metrics.

## Key Features
- Add and manage food items with nutritional information.
- Track daily measurements: weight, water intake, activity.
- Calculate daily totals for calories and macronutrients.
- Display charts and graphs for weekly or monthly trends.
- Store a history of entries for long-term tracking.

## Technologies Used
- **SwiftUI** for building the user interface.
- **Core Data** for local data storage.
- **Swift Charts** for visualizing nutrition and measurement trends.
- Compatible with **iOS 15+**.

## Additional Notes
- Simple and intuitive interface for easy daily use.
- Designed for personal health tracking and nutrition awareness.
